# BigEyedAl

A gem that prints out a different phrase depending on the string passed.

## Installation

Add this line to your application's Gemfile:

    gem 'big_eyed_al'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install big_eyed_al

## Usage

- run IRB/Pry
- require 'big_eyed_al'
- run BigEyedAl.said(string)
